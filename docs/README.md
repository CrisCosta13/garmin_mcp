# Trainer OS

Documentação do sistema de análise de performance da atleta.

## Estrutura

- `core/` — regras centrais e processo de decisão
- `athlete/` — perfil da atleta
- `garmin/` — interpretação dos dados Garmin
- `training/` — CrossFit, musculação e análise de carga
- `competition/` — provas, estratégias e preparação
- `reports/` — relatórios e fechamentos
